# プロフィールを表示 {#show-profiles}

`SHOW PROFILES`ステートメントは現在、空の結果のみを返します。

## あらすじ {#synopsis}

**表示ステートメント:**

![ShowStmt](/media/sqlgram/ShowStmt.png)

## 例 {#examples}

{{< copyable "" >}}

```sql
SHOW PROFILES;
```

```
Empty set (0.00 sec)
```

## MySQL の互換性 {#mysql-compatibility}

このステートメントは、MySQL との互換性のためにのみ含まれています。 `SHOW PROFILES`を実行すると、常に空の結果が返されます。
